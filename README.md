# GenAI Assignment

This repository demonstrates a workflow of using a generative AI coding assistant (Claude) to draft content, then manually editing that content, while tracking all changes with Git and GitHub.

## Git Steps Used

1. Created a local project folder and initialized it with `git init`.
2. Created three files (`README.md`, `assistant_draft.md`, `final_version.md`) and made an initial commit.
3. Used Claude to generate content for `assistant_draft.md` and committed the result.
4. Copied the draft to `final_version.md`, made manual edits, and committed the edited version.
5. Updated this `README.md` and pushed all commits to GitHub.
